# Installation

### Install go sdk 1.10.4

See how to [install go](https://golang.org/doc/install#install)

```
cd $GOPATH
mkdir -p src/ctco-dev
cd src/ctco-dev
git clone https://github.com/ctco-dev/go-api-template.git
```

### Local run
```
cd src/ctco-dev/go-api-template
go run cmd/main.go
```

### Docker run 
```
./start.sh
```
